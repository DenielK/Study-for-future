1. В какой программе мы практикуем команды для работы с github на рабочих компьютерах?
- Git CMD

2. Что делают данные элементы кода #, ##, ###? (подсказка readme.md)
- # заголовок первого уровня
  ## заголовок второго уровня
  ### заголовок третьего уровня

3. Чем git push отличается от git pull?
- git push отправляет локальные изменения в удалённый репозиторий.

 git pull загружает изменения из удалённого репозитория в локальный.
